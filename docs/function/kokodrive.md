# ここドライブ (Koko Drive) ご利用ガイド

「ここドライブ (Koko Drive)」は、ここなつアカウントを作成することで利用できる便利なクラウドストレージです。
日常的なファイルの保管から、大切なデータの安全な保護、別ユーザーとのファイル共有まで簡単に行うことができます。

## 📁 1. ファイルとフォルダの管理
直感的な操作で、パソコン内と同じようにファイルを管理できます。

*   **簡単なアップロード**: 画面へのドラッグ＆ドロップ（D&D）で、複数ファイルや大容量ファイルもスムーズに一括アップロードできます。特に100MB以上の大容量ファイルは自動的に分割されて安定してアップロードされます。
*   **フォルダの整理**: フォルダの中にフォルダを作成し、ツリー構造でファイルを整理できます。
*   **ファイル操作**: ファイル名やフォルダ名の変更、移動などの操作にも対応しています。
*   **容量制限と確認**: ユーザーごとに使用可能なストレージ容量（クォータ）が定められています。現在の使用状況や残りの空き容量は画面上からいつでも確認でき、上限を超えるアップロードは自動で防止されます。
*   **安全な通信**: 何らかの理由でアップロードなどが途中で中断されてしまった場合でも、見えない一時ファイルが容量を圧迫し続けないよう、24時間経過後に自動でシステムがクリーンアップを行います。

## 🔒 2. 「金庫 (Vault)」による強力なデータ保護
誰にも見られたくない重要なファイルは、「金庫」に保管することで強力に保護されます。

*   **徹底した暗号化**: 「🔒 金庫」フォルダに入れたファイルは、ご自身以外の誰も中身を見ることができない仕組みにより強固に暗号化されます。
*   **マスターキーと復元**: ご自身だけの「マスターキー」を使ってファイルをロックします。万が一パスワードを忘れた場合のために、リカバリーフレーズでの復元もサポートしています。
*   **⚠️ 復元不可の厳格な仕様**: マスターキー本体のパスワードを忘れ、さらにリカバリーフレーズも紛失してしまった場合、**システム管理者を含め誰であっても金庫内のファイルを二度と復元できなくなります**（これにより絶対的な秘匿性が担保されています）。

## 🗑️ 3. 誤操作を防ぐ「ゴミ箱」機能
間違ってファイルを消してしまっても、すぐに救出できる安全設計です。

*   **一時的な保管（30日間）**: 削除したファイルやフォルダはすぐには消えず、一度「ゴミ箱」へ移動します。**ゴミ箱に移動してから30日が経過すると自動的に完全に削除されます**。
*   **いつでも復元**: 完全削除される前であれば、ゴミ箱の中にあるファイルはいつでも元の場所や指定の場所へ戻すことができます。

## 🔗 4. 安心・便利なファイル共有
アップロードしたファイルを他のユーザーと共有することができます。

*   **共有リンクの発行**: ファイルごとに共有専用のリンクURLを作成できます。
*   **セキュアな制限設定**: 
    *   **パスワード保護**: 知っている人だけがダウンロードできるようにパスワードを設定できます。
    *   **有効期限**: リンクの有効期限（時間）を設定し、期限切れ後はアクセスできないようにできます。
    *   **ダウンロード回数制限**: ダウンロードされる上限回数を決めることができます。
*   **直リンク防止と使い捨てURL**: 共有機能を使わずに自分用にファイルをダウンロードする際のURLは、セキュリティ上の理由から**「発行後5分間のみ有効な使い捨てリンク」**となっています。他人に無断でリンクを転載されてもファイルを守り抜く仕様です。

## 👁️ 5. 多彩なプレビュー機能と快適なUI
ファイルをダウンロードしなくても、その場で内容を確認できます。

*   **対応ファイル**: PDFドキュメント、テキストファイル、Markdown、各種プログラムコード（色分け付き）、フォント（文字の見た目確認）などをブラウザ上で直接プレビュー可能です。
*   **見やすいデザイン**: ダークモードとライトモードに対応しており、お好みのテーマで快適に操作いただけます。
*   **アップロード状況の確認**: 転送状況や残り時間などがプログレスバーで分かりやすく表示されます。
