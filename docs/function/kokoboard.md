# ここぼーど (Koko Board) ご利用ガイド

「ここぼーど (Koko Board)」は、ここなつアカウントをお持ちでなくても、Discordアカウントのみ（閲覧・検索の場合はログイン不要）で気軽にご利用いただける独自のDiscordサーバー紹介・検索掲示板です。
自分のサーバーの宣伝から、新しいコミュニティ探しまでを快適に行うための様々な機能が搭載されています。

## 📋 1. サーバーの掲載機能
ご自身が管理権限を持つDiscordサーバーを掲示板に掲載し、他のユーザーに向けて宣伝することができます。

*   **自動情報の取得**: サーバーのアイコン、サーバー名、総メンバー数、オンラインメンバー数などはDiscordから自動的に取得・同期されます。手動で入力する手間が省け、常に最新の状態が保たれます。
*   **詳細設定**: サーバーの「説明文」「カテゴリ」「タグ」を自由に設定できます。
*   **自動招待リンク**: 掲載時にボットが期限切れのない自動招待リンクを生成するため、招待URL切れの心配がありません。
*   **OGP画像の自動生成**: サーバーの独自ページURLをTwitter（X）やDiscordなどに共有した際、サーバーアイコンと名前が綺麗に合成された「OGP画像」が自動生成されて表示されます。

## 🚀 2. アップ（Bump）機能
掲示板内で自分のサーバーを目立たせるための必須機能です。

*   **上位表示**: Discord内で`/up`コマンドを実行すると、掲示板の標準の並び順（新着アップ順）で自分のサーバーが最上位に表示されます。
*   **クールダウン**: `/up`コマンドは**2時間**に1回実行可能です。
*   **ランダム応援アップ**: `/randomup`コマンドを使用すると、掲示板内の他サーバーをランダムでアップして応援できます。この機能は12時間に1回利用でき、応援のお礼として**100ここ**が報酬として付与されます。
*   **記念メッセージ**: アップに成功すると、これまでの「総アップ回数」「掲載開始からの日数」「1日あたりの平均アップ回数」などの統計情報がDiscord上に通知されます。
*   **区切りのお祝い**: 1回目、10回目、100回目、500回目、1000回目など、記念すべきアップ回数に到達した際には特別なメッセージでお祝いされます。継続的なアップがサーバーの信頼度にも繋がります。

## 🔍 3. 検索とソート表示機能
目的のサーバーを簡単に見つけることができます。

*   **キーワード＆タグ検索**: サーバー名、説明文、登録されているタグを使ってキーワード検索が可能です。
*   **人気のタグ**: 掲示板内でよく使われているタグ（トップ10）が一覧表示され、流行りのカテゴリからサーバーを探せます。
*   **多彩な並び替え**: 
    1.  **アップ順**（最近アップされた活発なサーバー順・デフォルト）
    2.  **評価順**（レビューの星の平均が高い順）
    3.  **レビュー数順**（口コミの件数が多い順）
    4.  **新着順**（最近掲示板に登録された順）

## ⭐ 4. レビュー（評価）機能
参加したサーバーに対して、感想を書き込むことができます。

*   **5段階評価とコメント**: 1〜5の星（レーティング）と、テキストでのコメントを残すことができます。
*   **情報の透明性**: レビューには投稿者のDiscordアイコンとユーザー名が表示されます。
*   **管理者の仕様**: 自作自演を防ぐため、**サーバーの管理者自身は自分のサーバーにレビュー（評価）を直接作成することはできません**。
*   **コミュニケーション**: ユーザーから投稿されたレビューに対して「いいね」を付けたり、管理者が「返信（リプライ）」を書き込むことができ、双方向のコミュニケーションが可能です。

## ❤️ 5. お気に入り機能
気になるサーバーや、よく利用するサーバーを保存しておけます。

*   **お気に入り登録**: サーバー詳細ページからワンクリックで「お気に入り」に追加できます。
*   **専用ページ**: 登録したサーバーは「お気に入り一覧ページ」からいつでもすぐに確認できます。

## 👑 6. プレミアム枠システム
掲示板の最も目立つ特等席（プレミアムスロット）を獲得するためのオークションシステムです。

*   **入札システム**: 経済機能の仮想通貨（ここ）を使用して、プレミアム枠に入札することができます。
*   **競り上げ式**: 誰かが入札している場合、次の人は現在の価格よりも高く（最低でも10%以上高く）入札する必要があります。
*   **30日間の独占**: 落札が確定すると、獲得した時点から**30日間**、そのサーバーがプレミアム枠として掲示板の最上位等に固定で表示され続けます。
*   **履歴の公開**: 過去のプレミアム枠の落札額と、どのサーバーがいくらで落札したか等の履歴情報は誰でも確認できるようになっています。

## 🛡️ 7. その他の仕様・自動保護
*   **不正な公開の防止**: ボットをサーバーからキック（追放）した場合、そのサーバーの掲示板への掲載は**自動的に非公開**状態に切り替わります。（再度公開するにはボットの再導入が必要です）
*   **プレミアム枠の没収**: プレミアム枠の権利を持っている状態でボットをキックした場合は、そのプレミアム権も即座に消滅・没収される仕様となっています。
