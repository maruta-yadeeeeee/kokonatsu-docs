# ここなつeconomy (Kokonatsu Economy)

ここなつeconomyは、サーバー全体で共通して使える架空の通貨を通じた経済シミュレーション機能です。
お金を稼ぎ、アイテムを買い、投資をし、他のユーザーと経済力を競い合うことができます。

---

## 1. 基本的な通貨
*   **ここ (kk)**: 基本となる現金（キャッシュ）です。給料やカジノの配当などはすべて「kk」で支払われます。
*   **こここいん (kkc)**: 変動相場制の暗号資産（仮想通貨）です。「kk」を使って売買でき、価格の上下によって利益（または損失）を生み出します。

## 2. 資産と税金
あなたの総資産は、「現金 (kk) + 保有こここいんの時価 + 保有ペットの価値 + アイテムの価値」の合計で計算されます。
ここなつeconomyでは、富の集中を防ぐため、**総資産額に応じた累進課税制度（所得税）**が自動的に適用されます。給料などを受け取る際、この税率分が天引きされます。

*   **10,000 kk 未満**: 0% (非課税)
*   **10,000 kk 以上**: 2%
*   **50,000 kk 以上**: 4%
*   **100,000 kk 以上**: 6%
*   **250,000 kk 以上**: 8%
*   **500,000 kk 以上**: 10%

## 3. 役職（ランク）と給与
毎日受け取れる基本給与やペットの所持上限は、あなたの「役職（ランク）」によって決まります。
上の役職に昇進するには、規定のジョブ回数をこなし、一定額の「kk」をバーン（焼却=消費）する必要があります。一部の役職には、ここなつアカウントとの連携（Webログイン）が必須条件となるものもあります。

1.  **Rank1(研修生)**: 給与 150〜250 kk / ペット上限 30匹
2.  **Rank2(一般職)**: 給与 200〜350 kk / ペット上限 35匹 (アカウント連携必須)
3.  **Rank3(専門職)**: 給与 250〜450 kk / ペット上限 40匹
4.  **Rank4(経営職)**: 給与 350〜550 kk / ペット上限 45匹
5.  **Rank5(国家顧問)**: 給与 500〜750 kk / ペット上限 50匹

## 4. ショップとアイテム
稼いだ「kk」を使って、Web上のショップから様々なアイテムを購入できます。通常ショップのほかに、ランダムで割引セットなどが並ぶ「デイリーショップ」が存在します。
*   **💊 寿命延長薬 (10,000 kk)**: 指定したペットの寿命を7日間延ばします。
*   **🛡️ 防犯ブザー (2,000 kk)**: 強盗（他ユーザーからの資金奪取）を防ぐ使い捨ての防御アイテムです。

## 5. こここいん市場
こここいんは、UniswapなどのAMM（自動マーケットメーカー）モデルを採用した本格的な仮想通貨市場です。
*   市場には「kk」と「kkc」の流動性プールがあり、**誰かが購入や売却を行うたびにリアルタイムで即座に価格が変動**します（大きな注文が入るほど大きく価格が動きます）。
*   価格推移の履歴（チャートデータ）は5分ごとに記録・保存され、コマンドから過去30日間の価格チャートとして確認できます。
*   売買には **最大 10% の取引手数料** がかかります。そのため、短期的な売買よりも、中長期的な大きな価格変動を狙うのが基本となります。

## 6. 強奪 (Rob) システム
他のユーザーの資産を狙って「強奪（`/rob`）」を仕掛けることができる、ハイリスク・ハイリターンなシステムです。
*   **実行制限**: 強奪は **24時間に1回** しか実行できません。
*   **強奪額と成功率**: 相手の「総資産」の特定の割合の範囲でランダムな額を狙います。狙う額の割合が大きいほど、強奪の成功率が下がる仕組みです。
*   **成功時の処理**: 相手の現金が足りない場合、システムが相手のペットや資産を「強制清算（売却）」して無理やり現金をひねり出し、あなたに支払われます。
*   **失敗時のペナルティ**: 強奪に失敗した場合、逆に**あなた自身の総資産の特定の範囲でランダムな額の「罰金」を支払う**ことになります。この罰金はシステムに没収されます。
*   **防御手段**:
    1.  **防犯ブザー**: ショップで「🛡️ 防犯ブザー」を買っておくと、強奪された際に自動で消費され、強奪を**100%無効化**して相手を追い返します（相手の24時間制限も消費させます）。
    2.  **国家顧問の直感**: Rank5（国家顧問）のユーザーは、防犯ブザーがなくても **5%の確率で強奪をパッシブ回避** する特殊能力を持っています。
