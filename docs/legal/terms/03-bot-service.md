# 利用規約 — 第3章 Botサービス

**最終更新日: 2026年4月8日**

---

## 第14条（Botサービスの概要）

1. ここなつBotは、Discordサーバーに導入して利用するDiscordアプリケーションです。
2. Botの利用には、DiscordサーバーへのBotの追加権限（「サーバーの管理」権限を持つアカウント）が必要です。
3. Botを導入したDiscordサーバーの管理者（以下「サーバー管理者」といいます）は、本規約に加え、Botの利用条件に同意したものとみなします。

---

## 第15条（Botの提供機能）

ここなつBotは以下の機能を提供します（サービスの変更・追加・廃止があり得ます）。

| 機能区分 | 内容 |
|---------|------|
| **経済システム** | 仮想通貨（kk/kkc）の管理、ランク、課税、ショップ、ペット |
| **ここぼーど** | Discordサーバーの掲示板登録・管理 |
| **グローバルチャット** | 複数サーバー間のグローバルチャット同期 |
| **ここカジノ** | 仮想通貨を使用したカジノゲーム |
| **スパム対策** | メッセージフィルタリング・メンション制限 |
| **入退室管理** | 入室・退室メッセージの設定 |
| **ロールパネル** | セルフロール選択パネルの作成 |
| **チケット** | サポートチケットの管理 |
| **ピン留め** | ピン留めメッセージの高度な管理 |
| **リバーシ** | リバーシゲーム |
| **汎用コマンド** | その他の汎用スラッシュコマンド |

---

## 第16条（ダッシュボード）

1. 本サービスのWebダッシュボードから、Botの各種設定をGUIで変更できます。
2. ダッシュボードの利用にはここなつアカウントへのログインが必要です。
3. ダッシュボードにアクセスできるのは、対象のDiscordサーバーにおいて「サーバーの管理」権限を持つメンバーに限ります。
4. Botの設定変更は運営が合理的な判断のもとで記録する場合があります。

---

## 第17条（スパム対策・セキュリティ機能）

1. Botはスパム行為・荒らし行為等の不正行為を自動検知する機能を持ちます。
2. スパム対策機能による自動処置（メッセージ削除・メンバーのタイムアウト等）の結果についての責任は、当該機能を有効化したサーバー管理者が負います。
3. スパム対策の閾値設定等はサーバー管理者の責任のもと行ってください。
4. 自動検知の誤検知による損害について、運営は責任を負いません。

---

## 第18条（サブアカウント検知）

1. 本サービスは、デバイス情報・行動パターン等を用いたサブアカウント（複数アカウントの不正保持）検知機能を有します。
2. 検知情報はサーバー管理者に通知・提供される場合があります。
3. 検知はベストエフォートであり、検知精度を保証するものではありません。
4. 誤検知による不利益について、運営は責任を負いません。

---

## 第19条（Botの利用制限・停止）

1. 運営は、以下の場合にサーバーへのBotサービス提供を制限・停止することがあります。
   - サーバー管理者または当該サーバーのメンバーが本規約に違反した場合
   - サービスに支障をきたす運用が確認された場合
   - サービスのメンテナンスや緊急対応が必要な場合
2. Botはシステムメンテナンス・障害・Discord API の制限等により予告なく一時的に利用できなくなる場合があります。
3. Botの利用停止による損害について、運営は責任を負いません。
