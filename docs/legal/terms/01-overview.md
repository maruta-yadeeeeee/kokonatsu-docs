# 利用規約 — 第1章 総則・定義・適用範囲

**最終更新日: 2026年4月8日**  
**施行日: 2026年4月8日**

---

## 第1条（目的）

本利用規約（以下「本規約」といいます）は、ここなつ（以下「運営」といいます）が提供するすべてのサービス（以下「本サービス」といいます）の利用条件を定めるものです。ユーザーは本規約に同意したうえで本サービスをご利用ください。

---

## 第2条（適用範囲）

1. 本規約は、本サービスを利用するすべてのユーザーに適用されます。
2. 本規約に加え、各サービスに個別の規約・ガイドラインが定められている場合は、それらも本規約と併せて適用されます。
3. 個別規約と本規約が矛盾・抵触する場合は、個別規約が優先して適用されます。

---

## 第3条（用語の定義）

本規約において使用する主な用語の定義は以下のとおりです。

**共通用語**

| 用語 | 定義 |
|------|------|
| **本サービス** | ここなつが提供するすべてのサービス（Bot、Webサービス、API等）の総称 |
| **ユーザー** | 本サービスを利用するすべての個人・法人 |
| **運営** | ここなつのサービスを運営・管理する主体 |
| **コンテンツ** | ユーザーが本サービス上に投稿・送信・保存するすべてのデータ |

**Discord関連**

| 用語 | 定義 |
|------|------|
| **Discordアカウント** | Discord Inc. が提供するアカウント |
| **ここなつBot** | ここなつが提供するDiscordアプリケーション |
| **ここぼーど** | ここなつBotと連携するDiscordサーバー掲示板サービス |
| **グローバルチャット** | ここなつBotを通じて複数のDiscordサーバー間でメッセージを同期する機能 |

**独自サービス**

| 用語 | 定義 |
|------|------|
| **ここなつアカウント** | 本サービスへのログインに使用する統合アカウント（Discord SSOまたはメール/パスワードで利用） |
| **kk** | 本サービス内で使用する仮想通貨（法定通貨への換金不可） |
| **kkc** | 本サービス内のAMM型仮想資産（法定通貨への換金不可） |
| **ここドライブ** | ここなつが独自に提供するクラウドストレージサービス |
| **ここカジノ** | 仮想通貨（kk）を使用したここなつ独自のカジノゲームサービス |
| **OAuth2連携** | ここなつアカウントを外部サービスへ連携するOAuth2/OIDC機能 |

---

## 第4条（年齢制限）

1. 本サービスは **13歳以上** の方を対象としています。
2. 本サービスはDiscordと連携するため、Discord利用規約が定める年齢要件も遵守してください。
3. 13歳未満の方の利用が判明した場合、運営はアカウントを停止・削除することができます。
4. 18歳未満の方が本サービスを利用する場合は、保護者の同意を得たうえでご利用ください。

---

## 第5条（規約への同意）

1. ユーザーは、本サービスの利用を開始した時点で本規約に同意したものとみなします。
2. 本規約に同意しない場合は、本サービスをご利用になれません。
3. 法人・団体としてご利用の場合は、適正な権限を有する代表者が同意するものとします。

---
