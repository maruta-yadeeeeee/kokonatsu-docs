# 利用規約 — 第10章 禁止事項

**最終更新日: 2026年4月8日**

---

## 第56条（禁止事項の全般）

ユーザーは本サービスを利用するにあたり、以下の行為を行うことを禁止します。

---

### 1. アカウント・なりすましに関する禁止事項

| 禁止行為 | 説明 |
|---------|------|
| **複数アカウントの保持** | 1人が複数のここなつアカウントを作成・使用すること（運営が別途認めた場合を除く） |
| **アカウントの譲渡・共有** | 他人にアカウントを貸与・譲渡・売買すること |
| **なりすまし** | 他のユーザー・運営・著名人等になりすますこと |
| **虚偽情報の登録** | 他人のメールアドレスや虚偽の情報を用いてアカウントを作成すること |
| **不正ログイン** | 他人のアカウントに不正にアクセスしようとすること |

---

### 2. セキュリティ・システムに関する禁止事項

| 禁止行為 | 説明 |
|---------|------|
| **不正アクセス** | 本サービスのシステムへの不正アクセス・侵入試行 |
| **脆弱性の悪用** | バグ・脆弱性を発見した場合の悪用（発見した場合は運営へ報告してください） |
| **自動化ツールの不正使用** | スクレイパー・ボット・自動化スクリプト等を用いたサービスへの大量アクセス |
| **リバースエンジニアリング** | 本サービスのソースコード・アルゴリズムを不正に解析・複製すること |
| **DDoS・DoS攻撃** | サービスに対する過負荷攻撃 |

---

### 3. コンテンツに関する禁止事項

| 禁止行為 | 説明 |
|---------|------|
| **違法コンテンツの投稿・保存** | 著作権侵害・違法薬物・マルウェア等の違法コンテンツ |
| **CSAM** | 児童性的虐待に関するコンテンツの一切（発見次第、関係機関に通報します） |
| **プライバシー侵害** | 他者の個人情報・プライベートな写真・動画の無断公開 |
| **ヘイトスピーチ** | 人種・民族・性別・宗教・国籍・障害等に基づく差別的・侮辱的な発言 |
| **脅迫・ハラスメント** | 他者への脅迫・ストーキング・嫌がらせ行為 |

---

### 4. 経済システムに関する禁止事項

| 禁止行為 | 説明 |
|---------|------|
| **仮想通貨の換金** | kk・kkc等の仮想通貨を法定通貨・実物価値のあるものと交換すること |
| **不正なkk取得** | バグ・脆弱性・自動化ツールを利用したkkの不正取得 |
| **カジノの不正行為** | 自動化・外部ツール等を使用した不正なゲームプレイ |
| **経済操作** | kkcプールを意図的に操作する行為 |

---

### 5. その他の禁止事項

| 禁止行為 | 説明 |
|---------|------|
| **スパム** | 同一内容の大量送信・無差別勧誘・フィッシングリンクの流布 |
| **無許可の広告** | 運営の許可なく商業的広告・勧誘を行うこと |
| **サービスの妨害** | 他のユーザーのサービス利用を妨害する行為 |
| **運営への偽情報の提供** | 問い合わせ・報告において虚偽の情報を提供すること |
| **Discord利用規約違反** | Discordの利用規約・コミュニティガイドラインに違反する行為 |

---

## 第57条（違反への対応）

禁止事項に違反した場合、運営は以下の措置を講じることがあります。

1. 警告の発令
2. コンテンツ・投稿の削除
3. 特定機能の利用制限
4. アカウントの一時停止
5. アカウントの永久停止・削除
6. 不正に取得した仮想通貨・アイテムの没収
7. 関係機関への通報

措置の内容は違反の重大性・頻度等を考慮して決定します。重大な违反（特に CSAM・不正アクセス・脅迫等）については、事前通知なく最も厳しい措置を講じます。

---

## 第58条（脆弱性の報告）

サービスのバグ・セキュリティ上の脆弱性を発見した場合は、悪用することなく、速やかに運営までご報告ください。誠意ある報告に感謝します。
