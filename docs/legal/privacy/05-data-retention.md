# プライバシーポリシー — 保存期間・削除

**最終更新日: 2026年4月8日**

---

## 基本方針

運営は、収集した情報を**サービス提供に必要な期間のみ**保存します。目的を達成した情報は適切に削除または匿名化します。

---

## 各データの保存期間

### アカウント情報

| データ種別 | 保存期間 | 備考 |
|-----------|---------|------|
| アカウント基本情報（メール・パスワードハッシュ等） | アカウント有効期間中 | アカウント削除後は後述の方針に従う |
| DiscordユーザーID・連携情報 | アカウント有効期間中 | |
| MFAシークレット | MFA無効化またはアカウント削除まで | |
| ログイン履歴 | 最大 **90日間** | セキュリティ監査目的 |
| セッション情報 | セッション終了後 **30日間** | 不正アクセス検知目的 |

---

### ここドライブ

| データ種別 | 保存期間 | 備考 |
|-----------|---------|------|
| 通常ファイル | ユーザーが削除するまで | ゴミ箱移動後30日で完全削除 |
| ゴミ箱内ファイル | 移動後 **30日間** | 期間後に完全削除 |
| 共有リンク情報 | リンク有効期限まで、または削除まで | |
| 使い捨てURL | 発行から **5分間** | 自動無効化 |

---

### 通信・ログ情報

| データ種別 | 保存期間 | 備考 |
|-----------|---------|------|
| グローバルチャットメッセージ | 送信後 **最大90日間** | ログ・不正対応目的 |
| アクセスログ | **30日間** | セキュリティ・障害対応目的 |
| GeoIP・デバイス情報 | **90日間** | セキュリティ目的 |
| デバイスフィンガープリント | **180日間** | サブアカウント検知目的 |

---

### 経済システム

| データ種別 | 保存期間 | 備考 |
|-----------|---------|------|
| 仮想通貨残高 | アカウント有効期間中 | |
| 取引履歴 | **1年間** | 不正取引の調査・統計目的 |
| カジノゲーム履歴 | **90日間** | 公正性検証目的 |

---

### OAuth2情報

| データ種別 | 保存期間 | 備考 |
|-----------|---------|------|
| アクセストークン | 発行から有効期限まで | |
| リフレッシュトークン | ユーザーが連携を解除するまで | |
| 認可コード | 発行から **10分間** | 使用または期限切れで無効化 |

---

## アカウント削除後の取り扱い

ユーザーがアカウントを削除した場合、以下の方針に従います。

1. **即時削除・無効化**: セッション情報・OAuth2トークン・共有リンク
2. **30日以内に削除**: アカウント基本情報（メール・パスワードハッシュ・MFAシークレット）、ここドライブのファイル
3. **保持期間経過後に削除**: ログ情報・取引履歴（各保存期間に従う）
4. **匿名化して統計に保持**: 個人を特定できる情報を除いた統計データ

---

## 例外的な保持

以下の場合、通常の保存期間を超えてデータを保持することがあります。

- **法的紛争・調査**: 裁判所命令・捜査機関の要請がある場合
- **規約違反の証拠保全**: 重大な規約違反の証拠として保全が必要な場合
- **法的義務**: 法律上の保存義務がある場合

---

## データの削除方法

ユーザー自身によるデータ削除は以下の方法で行えます。

| 操作 | 削除されるデータ |
|------|--------------|
| アカウント設定 > アカウント削除 | アカウントに紐づく全データ（上記方針に従い段階的に削除） |
| ドライブ > ファイル削除 | 対象ファイル（ゴミ箱経由で30日後に完全削除） |
| ドライブ > ゴミ箱を空にする | ゴミ箱内ファイルを即時完全削除 |
| アカウント設定 > 連携アプリ > アクセス取り消し | 特定外部アプリのOAuth2トークン |
| アカウント設定 > セッション管理 > ログアウト | 指定セッション |
