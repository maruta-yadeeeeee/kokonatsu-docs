# プライバシーポリシー — 利用目的

**最終更新日: 2026年4月8日**

---

## 個人情報の利用目的

収集した情報は、以下の目的のためにのみ使用します。**収集した情報を目的外に利用することはありません。**

---

## 1. サービスの提供・運営

| 利用目的 | 対象データ |
|---------|----------|
| ここなつアカウントの認証・管理 | アカウント情報、セッション情報 |
| Discord Bot機能の提供 | DiscordユーザーID、サーバー情報 |
| ここドライブのファイル管理機能の提供 | ファイルデータ、メタデータ |
| 経済システム（kk・kkc・ペット）の運営 | 残高・取引履歴 |
| ここぼーどの掲示板機能の提供 | 投稿コンテンツ |
| グローバルチャットのメッセージリレー | メッセージ内容・メタデータ |
| OAuth2連携機能の提供 | アカウント情報（同意済みスコープ分） |

---

## 2. セキュリティ・不正防止

| 利用目的 | 対象データ |
|---------|----------|
| 不正ログイン・アカウント乗っ取りの検知 | IPアドレス、デバイス情報、ログイン履歴 |
| 新規デバイスからのログイン通知 | IPアドレス、GeoIP情報、ユーザーエージェント |
| サブアカウント（複数アカウント）の検知 | デバイスフィンガープリント、行動パターン |
| VPN・Proxy・Torノードからのアクセス検知 | IPアドレス |
| スパム・荒らし行為の検知 | メッセージ内容、送信パターン |
| カジノの公正性確保 | ゲーム履歴、シード情報 |
| 禁止コンテンツの通報対応 | ファイルメタデータ（金庫を除く） |

---

## 3. ユーザーサポート・通知

| 利用目的 | 対象データ |
|---------|----------|
| パスワードリセット・MFAリカバリーのメール通知 | メールアドレス |
| 新規デバイスログインの通知メール | メールアドレス、GeoIP情報 |
| サービスに関する重要なお知らせの配信 | メールアドレス |
| 規約違反への対応・通知 | アカウント情報 |

---

## 4. サービス改善・統計分析

| 利用目的 | 対象データ |
|---------|----------|
| サービス全体の統計情報の収集・分析 | 利用状況（個人を特定しない形式） |
| 障害・バグの検知と修正 | エラーログ、アクセスログ |
| 新機能の開発・改善 | 利用パターン（個人を特定しない形式） |

> 統計分析においては、個人を識別できない形式（匿名化・集計済みデータ）で行うよう努めます。

---

## 5. 法的義務の履行

| 利用目的 | 対象データ |
|---------|----------|
| 法的要請・行政命令への対応 | 必要な範囲の情報 |
| 不正行為の証拠保全 | 関連するログ・記録 |

---

## 目的外利用の禁止

上記以外の目的で個人情報を利用する場合は、事前にユーザーの同意を取得します。ただし、以下の場合を除きます。

- 法令に基づく場合
- 人の生命・身体・財産の保護のために必要であり、本人の同意を得ることが困難な場合
- 公衆衛生の向上または児童の健全な育成推進のために特に必要な場合
