# プライバシーポリシー — 収集する情報

**最終更新日: 2026年4月8日**

---

## 収集する情報の概要

本サービスは、サービスの提供・安全性の確保・品質向上のために、以下の情報を収集します。

---

## 1. アカウント情報

### 1-1. Discordアカウント情報

Discord連携でログインした場合、Discord APIを通じて以下の情報を取得します。

| 情報 | 取得タイミング | 用途 |
|------|-------------|------|
| DiscordユーザーID | ログイン時 | アカウントの識別 |
| ユーザー名（グローバル名含む） | ログイン時 | プロフィール表示 |
| アバター画像URL | ログイン時 | プロフィール表示 |
| 所属するDiscordサーバー一覧 | ダッシュボードアクセス時 | 管理対象サーバーの確認 |

### 1-2. メール/パスワード認証情報

メールアドレスとパスワードで登録した場合、以下の情報を収集します。

| 情報 | 保存方法 | 用途 |
|------|---------|------|
| メールアドレス | 平文（SHA-256ハッシュでインデックス） | ログイン・通知の送信 |
| パスワード | bcrypt等のハッシュ関数でハッシュ化 | 認証 |
| メール認証トークン | ハッシュ化 | メールアドレスの確認 |

### 1-3. 多要素認証（MFA）情報

| 情報 | 保存方法 | 用途 |
|------|---------|------|
| TOTPシークレット | 暗号化して保存 | MFA認証 |
| リカバリーコード | ハッシュ化 | MFAリカバリー |

---

## 2. ネットワーク・デバイス情報

### 2-1. ログイン・セッション情報

| 情報 | 用途 |
|------|------|
| IPアドレス | セッション管理・不正アクセス検知 |
| ユーザーエージェント（ブラウザ・OS情報） | デバイス識別・セキュリティ通知 |
| セッショントークン | ログイン状態の維持 |
| ログイン日時 | セキュリティ監査・ログイン履歴 |

### 2-2. GeoIP情報

IPアドレスから以下の地理情報を推定します。この情報は**統計情報・セキュリティ目的**にのみ使用します。

- 国・地域
- 都市（おおよそ）
- タイムゾーン
- ISP（インターネットサービスプロバイダ）

### 2-3. デバイスフィンガープリント

サブアカウントの検知等のセキュリティ目的のために、ブラウザ・デバイスの特性を組み合わせたフィンガープリントを収集・記録することがあります。

---

## 3. サービス利用情報

### 3-1. 経済システム

| 情報 | 用途 |
|------|------|
| 仮想通貨（kk・kkc）の残高・取引履歴 | 経済システムの運営 |
| ショップ購入履歴 | 購入アイテムの付与 |
| カジノゲーム履歴 | 公正性検証・統計 |
| ランク・経験値 | ランクシステムの運営 |
| ペット情報 | ペットシステムの運営 |

### 3-2. ここドライブ

| 情報 | 用途 |
|------|------|
| アップロードしたファイル（内容・メタデータ） | ストレージサービスの提供 |
| フォルダ構造 | ファイル管理機能の提供 |
| 共有リンク設定 | 共有機能の提供 |
| ストレージ使用量 | 容量管理 |

:::caution 金庫機能について
金庫に保存されたファイルはエンドツーエンド暗号化されており、運営であっても内容を確認することはできません。
:::

### 3-3. ここぼーど

| 情報 | 用途 |
|------|------|
| レビュー・評価の内容 | 掲示板サービスの提供 |
| Bump履歴 | Bump機能の管理 |

### 3-4. グローバルチャット

| 情報 | 用途 |
|------|------|
| 送信したメッセージの内容 | グローバルチャット機能の提供 |
| 送信メッセージのメタデータ（送信時刻・サーバー情報） | ログ管理・不正行為対応 |

### 3-5. OAuth2連携

| 情報 | 用途 |
|------|------|
| 登録された外部アプリ情報 | OAuth2サービスの提供 |
| 発行されたアクセストークン・リフレッシュトークン | 認証状態の管理 |
| ユーザーが付与したスコープ（権限） | 許可されたデータの提供 |

---

## 4. 情報収集を行わないケース

以下の情報は収集しません。

- クレジットカード・銀行口座番号などの決済情報（本サービスは有料サービスを提供していません）
- 金庫内ファイルの内容（エンドツーエンド暗号化のため）
- 本サービス外での行動情報
